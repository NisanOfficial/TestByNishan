🚀 Setup & Run

1. Clone / Download
```bash

cd NisanX
```
2. Logo Add Koro
`NisanX.jpg` file ta root folder e rakho. Size: 200x200px recommend.

3. Telegram Setup (Optional)
`index.html` er moddhe line 520-521:
```js

const CHAT_ID = 'YOUR_CHAT_ID_HERE';     // @userinfobot theke niba
```
